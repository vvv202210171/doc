C2C 商户管理接口

1. 创建商户

请求方法
POST
请求地址
/c2c/merchant/create
权限要求
需要认证

描述

用户创建 C2C 商户

请求参数

参数 类型 必填 说明
name string ✓ 商户全称，不能重复
phone string ✓ 商户联系手机号
type string 可选 商户类型：PERSONAL 或 PLATFORM，默认 PERSONAL

请求示例

curl --location --request POST 'https://m.biyiex.top/api/c2c/merchant/create' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data-raw '{
  "name": "我的商户",
  "phone": "13800138000",
  "type": "PERSONAL"
}'

响应示例（成功）

{
  "code": 0,
  "msg": "success",
  "data": {
    "id": 1,
    "merchantCode": "MCH20260302001",
    "name": "我的商户",
    "member": "U10001",
    "type": "PERSONAL",
    "status": "PENDING",
    "phone": "13800138000",
    "verifiedStatus": 1
  }
}

C2C 收款方式接口

7. 创建收款方式

请求方法
POST
请求地址
/c2c/collway/create
权限要求
需要认证

描述

为商户添加收款方式（微信、支付宝、银行卡）

请求参数 - 微信

参数 类型 必填 说明
collWay string ✓ wx
wxUrl string ✓ 微信收款二维码 URL
wxAcc string ✓ 微信账户/微信号
wxName string ✓ 微信真实姓名
tag string 可选 标记/备注

微信请求示例

curl --location --request POST 'https://m.biyiex.top/api/c2c/collway/create' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data-raw '{
  "collWay": "wx",
  "wxUrl": "https://example.com/wx_qrcode.png",
  "wxAcc": "wechat_account_123",
  "wxName": "张三",
  "tag": "主账户"
}'

请求参数 - 支付宝

参数 类型 必填 说明
collWay string ✓ alipay
alipayUrl string ✓ 支付宝收款二维码 URL
alipayAcc string ✓ 支付宝账户
alipayName string ✓ 支付宝真实姓名

请求参数 - 银行卡

参数 类型 必填 说明
collWay string ✓ bank
bankName string ✓ 开户银行名称
bankAcc string ✓ 银行账户号
bankHolder string ✓ 持卡人姓名
bankAddr string 可选 开户行地址

银行卡请求示例

curl --location --request POST 'https://m.biyiex.top/api/c2c/collway/create' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data-raw '{
  "collWay": "bank",
  "bankName": "中国工商银行",
  "bankAcc": "6222022409001234567",
  "bankHolder": "张三",
  "bankAddr": "北京市朝阳区支行",
  "tag": "工行账户"
}'

🔐 认证方式

所有需要认证的接口都需要在请求头中添加：

Authorization: Bearer <your_token>

其中 your_token 是通过登录接口获取的 token。

⚠️ 错误响应

所有失败响应都遵循以下格式：

{
  "code": -1,
  "msg": "error_message",
  "data": null
}

HTTP 状态码

状态码 说明
200 请求成功
400 参数错误或业务验证失败
401 未授权（token 过期或无效）
500 服务器内部错误

📋 字段值说明

商户类型（type）

值 说明
PERSONAL 个人商户（默认）
PLATFORM 平台商户

商户状态（status）

值 说明
PENDING 待审核
ACTIVE 已激活
SUSPENDED 已暂停
REJECTED 已拒绝

收款方式（collWay）

值 说明
wx 微信支付
alipay 支付宝
bank 银行卡转账
